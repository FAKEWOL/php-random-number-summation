# do-while-5-20-sayilar

## Description
This GitHub repository contains a simple PHP script that generates random numbers between 0 and 20 using a `do-while` loop until a number other than 5 is generated. Once a number other than 5 is obtained, the script calculates and prints the sum of all previously generated numbers.

## Features
- Generates random numbers between 0 and 20.
- Uses a `do-while` loop to ensure at least one iteration.
- Calculates the sum of generated numbers until a number other than 5 is encountered.
- Prints the final sum.

## Installation
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/do-while-5-20-sayilar.git
   ```
2. Navigate to the project directory:
   ```bash
   cd do-while-5-20-sayilar
   ```

## Usage
To run the script, simply execute it using a PHP interpreter:
```bash
php 5-20.php
```
The script will output the sum of all generated numbers until a number other than 5 is encountered.
